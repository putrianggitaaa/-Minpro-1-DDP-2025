# Sistem Informasi Keanggotaan Organisasi BEM FT 2025
Merupakan sebuah sistem informasi yang bisa memanajemen daftar anggota dari organisasi BEM FT. Di dalamnya terdapat 4 menu yang menggunakan konsep CRUD (Create, Read, Update, Delete). Sistem Informasi ini akan memudahkan pendataan anggota organisasi sehingga menjadi lebih efisien. Berikut adalah alur program dari Sistem Informasi Keanggotaan Organisasi BEM FT 2025.

# Menu 1: Menambah nama anggota baru menggunakan konsep CREATE
**Kondisi 1:** 
Di Menu 1, kita bisa menambahkan anggota baru ke dalam list keanggotaan BEM FT dengan mengetikkan nama dan jabatan sesuai format yang ditentukan. Nama anggota baru akan langsung ditampilkan di dalam list keanggotaan BEM FT setelah kita menambahkannya.\

**Kondisi 2:**
Apabila kita memasukkan nama dan jabatan anggota yang sudah ada sebelumnya, sistem ini tidak akan bisa menambahkannya lagi. \
